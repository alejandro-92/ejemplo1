** Cambiando de nombre a un repositorio remoto **
*
*-------------------------------------
*|   git remote add [nombre] [url]   |
*-------------------------------------
*
*Esto facilita el no tener que escribir toda la url de repositorio:
*
* normalmente se hace: git clone [url]
*
* con el nombre cambiado: git clone [name]
